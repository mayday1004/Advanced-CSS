### 在這些項目中我使用:

- SCSS 做為預處理器
- CSS 架構遵循 BEM 方法
- 響應式設計
- 針對螢幕分辨率做圖片切換，以提高效能

### 課堂以外的挑戰:

我在 Trillo 項目中添加了一些額外的部分:

- 個人及消息 icon 處新增選單功能
- 圖片展示區增加懸停效果
- 搜索欄添加搜索建議
